Detailed Description

template<typename T_Scalar, typename T_Attribute = uint64_t>
struct picongpu::adios::ReadNDScalars< T_Scalar, T_Attribute >

Functor for reading ND scalar fields with N=simDim In the current implementation each process (of the ND grid of processes) reads 1 scalar value Optionally the processes can also read an attribute for this dataset by using a non-empty attrName.

Template Parameters
T_ScalarType of the scalar value to read
T_AttributeType of the attribute (can be omitted if attribute is not read, defaults to uint64_t)
