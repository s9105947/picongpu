Detailed Description

template<class TYPE, unsigned DIM>
class pmacc::Exchange< TYPE, DIM >

Interface for a DIM-dimensional buffer used for data exchange.

Exchange defines an interface for exchanging data between hosts. Equally sized buffers are created on the device as well as on the host. Exchange buffers may use parts of existing GridBuffer or be used as dedicated memory.

Template Parameters
TYPEthe datatype for internal buffers
DIMthe dimension of the internal buffers
