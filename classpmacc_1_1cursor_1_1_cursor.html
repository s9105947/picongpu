Detailed Description

template<typename _Accessor, typename _Navigator, typename _Marker>
class pmacc::cursor::Cursor< _Accessor, _Navigator, _Marker >

A cursor is used to access a single datum and to jump to another one.

It is always located at a certain datum. Think of a generalized iterator.

Template Parameters
_AccessorPolicy functor class that is called inside operator*(). It typically returns a reference to the current selected datum.
_NavigatorPolicy functor class that is called inside operator()(). It jumps to another datum.
_MarkerRuntime data that is used by the accessor and the navigator. This is typically a data pointer.
