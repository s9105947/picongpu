template<uint32_t T_dim = simDim>
struct picongpu::IsPartOfSlice< T_dim >

Check if an offset is part of the slicing domain.

Check if a N dimensional local domain offset is equal to a scalar offset of a given dimension. The results can be taken to decide if a cell is within a slice of a volume.
