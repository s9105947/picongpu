Detailed Description

template<typename T_Scalar, typename T_Attribute = uint64_t>
struct picongpu::adios::WriteNDScalars< T_Scalar, T_Attribute >

Functor for writing ND scalar fields with N=simDim In the current implementation each process (of the ND grid of processes) writes 1 scalar value Optionally the processes can also write an attribute for this dataset by using a non-empty attrName.

Template Parameters
T_ScalarType of the scalar value to write
T_AttributeType of the attribute (can be omitted if attribute is not written, defaults to uint64_t)
