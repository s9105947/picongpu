template<typename T_MplSeq>
pmacc::AllCombinations< T_MplSeq >::type

Create tuples out of the elements of N sequences.

Combines all elements of N given sequences in T_MplSeq into N-tuples. If the number of elements in each sequence is S0, S1, ... S(N-1) than the resulting sequence will contain S0 * S1 * ... S(N-1) tuples.

example:

sequence == [ ] tuple == ( )

T_MplSeq = [[1,2],[1],[4,3]] combined to AllCombinations<T_MplSeq>::type = [(1,1,4),(1,1,3),(2,1,4),(2,1,3)]

Template Parameters
T_MplSeqN-dimensional sequence with input values or single type (e.g. bmpl::integral_c<uint32_t,5>) (if T_MplSeq is only one type it will be transformed to a sequence)
