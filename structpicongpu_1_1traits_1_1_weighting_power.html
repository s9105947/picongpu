template<typename T_Identifier>
struct picongpu::traits::WeightingPower< T_Identifier >

Describe if a particle attribute describes the quantity of a macro particle.

Depending on the implementation of an attribute, it might be sometimes useful to return a quantity regarding one of the underlying real particles (

See also
MacroWeighted).

This trait defines how each attribute needs to be scaled with the weighting (linear, quadratic, ...) to convert between "real" and "macro" particle attributes.
