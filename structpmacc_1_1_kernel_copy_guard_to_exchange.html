Detailed Description

template<uint32_t T_numWorkers>
struct pmacc::KernelCopyGuardToExchange< T_numWorkers >

copy particles from the guard to an exchange buffer

Warning
This kernel resets the number of particles in the processed supercells even if there are particles left in the supercell and does not guarantee that the last frame is contiguous filled. Call KernelFillGaps afterwards if you need a valid number of particles and a contiguously filled last frame.
