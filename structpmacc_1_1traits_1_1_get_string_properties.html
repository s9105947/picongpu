Detailed Description

template<typename T_Type>
struct pmacc::traits::GetStringProperties< T_Type >

get the properties of an object

The struct StringProperties<> needs to be specialized to change the result of this trait for a user defined type. If there is no user defined specialization available this trait inherits from the result of ::getStringProperties() from the queried type.
