Detailed Description

template<typename T_SpeciesType>
struct picongpu::particles::traits::GetIonizerList< T_SpeciesType >

Returns a sequence with ionizers for a species.

Several ionization methods can be assigned to a species which are called consecutively (in the same order as the user inputs them) within a single time step.
